Документация

Документация и возможности сборки и стартового шаблона

Запуск в режиме разработки: gulp

Сборка оптимизированной версии для продакшена: gulp build

Запуск конвертации шрифтов: gulp fonts

Запуск создания svg-spirite: gulp svg

Запуск создания favicon: gulp favicon

Возможности стартового шаблона

Готовые стили для CSS контейнеров разного типа.
Миксины и сниппеты для адаптива.
